## ⚙️ Installation
 
### Prerequisites
 
Make sure you have the following installed:
 
![Node.js](https://img.shields.io/badge/Node.js-v18+-339933?style=flat&logo=nodedotjs&logoColor=white)
![MongoDB](https://img.shields.io/badge/MongoDB-v6+-47A248?style=flat&logo=mongodb&logoColor=white)
![npm](https://img.shields.io/badge/npm-v9+-CB3837?style=flat&logo=npm&logoColor=white)
 
### Clone the Repository
 
```bash
# Clone this repository
git clone https://github.com/yourusername/bag-hub.git
 
# Navigate into the project directory
cd bag-hub
```
 
### Install Dependencies
 
```bash
npm install
```
 
### Setup Environment Variables
 
```bash
# Create .env file
cp .env.example .env
 
# Edit with your own values
nano .env
```
 
### Run the Application
 
```bash
# Development mode (with nodemon)
npm run dev
 
# Production mode
npm start
```
 
<div align="center">
 
> 🌐 App running at **http://localhost:3000**
 
</div>

## 🔑 Environment Variables
 
Create a `.env` file in the root directory:
 
```env
# Server
PORT=3000
NODE_ENV=development
 
# MongoDB
MONGODB_URI=mongodb://localhost:27017/baghub
 
# Session
SESSION_SECRET=your_super_secret_key
 
# (Optional) Cloudinary for image uploads
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```
