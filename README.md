## ⚙️ Installation

### Prerequisites

![Node.js](https://img.shields.io/badge/Node.js-v18+-1a3a6b?style=flat&logo=nodedotjs&logoColor=white)
![MongoDB](https://img.shields.io/badge/MongoDB-v6+-0e8b7a?style=flat&logo=mongodb&logoColor=white)
![npm](https://img.shields.io/badge/npm-v9+-f5a623?style=flat&logo=npm&logoColor=1a1f2e)

### Quick Start

```bash
# 1️⃣  Clone the repository
git clone https://github.com/Manjubs-123/BagCart-Ecommerce.git

# 2️⃣  Move into the project directory
cd BagCart-Ecommerce

# 3️⃣  Install all dependencies
npm install

# 4️⃣  Set up environment variables
cp .env.example .env

# 5️⃣  Start the development server
npm run dev
```

<div align="center">

> 🌐 Open **http://localhost:3000** in your browser to explore BagHub!

</div>

## 🗂️ Project Structure

```
BagCart-Ecommerce/
│
├── 📁 public/
│   ├── 📁 css/                  # Tailwind compiled styles
│   ├── 📁 js/                   # Client-side scripts
│   └── 📁 images/               # Static image assets
│
├── 📁 views/                    # EJS Template Files
│   ├── 📁 partials/
│   │   ├── header.ejs           # Navbar with BH logo
│   │   └── footer.ejs           # Dark navy footer
│   ├── index.ejs                # 🏠 Home — Hero + Carousels
│   ├── shop.ejs                 # 🛒 All products listing
│   ├── product-detail.ejs       # 🎒 Single product view
│   ├── cart.ejs                 # 🛒 Cart page
│   ├── checkout.ejs             # 💳 Checkout flow
│   ├── wishlist.ejs             # 💖 Saved items
│   ├── login.ejs                # 🔐 Login page
│   ├── signup.ejs               # 📝 Registration
│   └── admin/                   # ⚙️  Admin dashboard views
│
├── 📁 routes/
│   ├── auth.js                  # Login / signup / logout
│   ├── products.js              # Product CRUD routes
│   ├── cart.js                  # Cart management
│   ├── orders.js                # Order placement & history
│   └── admin.js                 # Admin-only routes
│
├── 📁 models/
│   ├── User.js                  # User schema (bcrypt)
│   ├── Product.js               # Product schema
│   ├── Order.js                 # Order schema
│   └── Cart.js                  # Cart schema
│
├── 📁 controllers/              # Business logic handlers
├── 📁 middleware/               # Auth guards, validators
├── 📁 config/
│   └── db.js                    # MongoDB connection
│
├── tailwind.config.js
├── .env
├── .gitignore
├── package.json
└── server.js                    # 🚀 App entry point
```

## 🔑 Environment Variables

```env
# ─── Server ──────────────────────────────────
PORT=3000
NODE_ENV=development

# ─── MongoDB ─────────────────────────────────
MONGODB_URI=mongodb://localhost:27017/baghub

# ─── Session ─────────────────────────────────
SESSION_SECRET=baghub_super_secret_key_2025

# ─── Cloudinary (Image Uploads) ──────────────
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```
